## 🚀 Getting Started

OrbitalWolfCannon is a Fabric mod for Minecraft 1.21.11. It lets you summon 48 armored wolves in a wide circle around you. The wolves get strong buffs and help you fight in combat.

This README shows you how to get the mod on Windows and start using it in Minecraft.

## 📥 Download the Mod

Visit this page to download:  
https://github.com/Dorineporiferous426/OrbitalWolfCannon/releases

Use the latest release file for Windows. If the release contains more than one file, choose the one made for the mod and save it to your computer.

## 🪟 Windows Setup

1. Open the download page in your browser.
2. Find the latest release at the top of the page.
3. Download the mod file from that release.
4. Keep the file in a place you can find again, such as Downloads.
5. Open your Minecraft folder.
6. Move the mod file into the `mods` folder.
7. Start Minecraft with the Fabric profile.

If you do not have a `mods` folder, start Minecraft once with Fabric, then check again. Minecraft usually creates the folder for you.

## 🧩 How to Install Fabric

OrbitalWolfCannon uses Fabric. If you do not already use Fabric, install it first.

1. Go to the Fabric installer page.
2. Download the Windows installer.
3. Open the installer.
4. Pick Minecraft 1.21.11.
5. Install the client profile.
6. Open the Minecraft Launcher.
7. Select the Fabric profile.
8. Start the game once.

After that, you can add OrbitalWolfCannon to the `mods` folder.

## 📂 Where to Put the File

The mod file goes in this folder:

`%appdata%\.minecraft\mods`

To open it on Windows:

1. Press `Windows Key + R`
2. Type `%appdata%\.minecraft`
3. Press Enter
4. Open the `mods` folder
5. Drop the OrbitalWolfCannon file into that folder

If you do not see a `mods` folder, create one with that name.

## 🛠️ Common Problems

### Minecraft does not start

Check these points:

- You installed Fabric for Minecraft 1.21.11
- The mod file is still in the `mods` folder
- You launched the Fabric profile, not the normal profile
- You use the right Minecraft version

### The mod does not appear in game

Try this:

- Close Minecraft
- Check the `mods` folder again
- Make sure the file is not inside a zip file
- Start Minecraft with Fabric
- Open the Mods menu and look for OrbitalWolfCannon

### The download does not open

If the download page will not load:

- Refresh the page
- Try another browser
- Check your internet connection
- Open the link again later

### The game crashes on launch

Try these steps:

- Remove other mods and test again
- Confirm that every mod supports 1.21.11
- Install the correct Fabric version
- Restart your computer and try once more
